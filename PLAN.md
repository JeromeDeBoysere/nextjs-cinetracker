# Project 2: CineTracker

## Description

Movie discovery application using the TMDB API: trends, search, genre filters, and detailed movie pages with modern design and high-quality images.

---

## Features

- Home page with popular, trending, and upcoming movies
- Movie search by title with debounce
- Genre filters (action, comedy, horror...)
- Pagination
- Movie detail page (poster, synopsis, cast, rating, runtime, release date)
- Favorites system (localStorage persistence)
- Responsive design

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS + shadcn/ui
- TMDB API
- Server Components
- Data fetching and caching (Next.js)
- ISR (Incremental Static Regeneration)
- Suspense and Streaming
- Zustand (state management)
- TanStack Query (data fetching and cache)
- Zod (API validation)

---

## Learning Objectives

- External API integration
- Caching strategies (revalidation)
- Server-Side Rendering (SSR)
- Optimized image handling (next/image)
- State management (Zustand + TanStack Query)
